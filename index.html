Uber Fare Prediction

The Uber Price Prediction Full Stack App predicts fare prices based on inputs like trip distance and vehicle type. Built with React.js (frontend) and Express.js (backend), it supports CRUD operations and integrates a machine learning model for accurate fare predictions. Data is stored in a database with real-time updates.

C-Shell

Implemented a custom shell program in C, capable of creating and managing new processes. It supports the execution of standard commands like cd, ls, echo, file redirection, and user-defined commands like pinfo. Technologies: C, OS, UNIX.

SimpleRA DataBase

Built a simplified relational database management system (DBMS) from the ground up using object-oriented principles in C++. The project supports essential database operations like load, sort, export, join, and group by, along with both assignment and non-assignment relational algebra operations. This project highlights proficiency in DBMS concepts and efficient data manipulation through custom-built algorithms. Technologies: C++
